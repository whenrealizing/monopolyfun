# Code of Conduct

MonopolyFun uses a concise community conduct policy based on respectful, evidence-driven collaboration.

## Expected Behavior

- Use clear, constructive language.
- Respect different experience levels and working styles.
- Keep discussion focused on the project and its users.
- Handle disagreements through evidence, tests, and concrete proposals.

## Enforcement Scope

Harassment, threats, personal attacks, private-information exposure, spam, abusive disruption, and misrepresented work or security evidence may lead maintainers to remove comments, reject contributions, or restrict participation.

## Reporting

Use GitHub Issues for ordinary community moderation concerns. Use the private security reporting path for issues involving vulnerabilities, account risk, credentials, or exploitable behavior.
