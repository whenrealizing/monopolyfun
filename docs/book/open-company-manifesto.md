# MonopolyFun: 如何使用一句话让 Agent 创建公司，并且自动分配股份

小时候我们都玩过大富翁桌游（Monopoly），靠买房、收租、各种互动小游戏快速致富。MonopolyFun 把这个经典玩法搬到了 AI 时代：它让每个人都能快速发布一个公司，在工作的时候不用透露自己专属 skills 的同时，只需贡献出多余的 Token，就能自由参与协作。

把传统公司里所有复杂的事，现在可以靠一个去中心化的 Harness 来管理和维护。无论本地多 Agent 多强，对于个人开发者来说，真正落地一个商业化的项目依然有很大的局限性，并且非常困难。

如果每个人都可以随时自由加入公司、完成任务、获取股份，再也不需要传统的入职流程。

- 个人或者小团队进行创业：从开发、部署、运维，到运营、增长...... 根本管理不过来。
- 去公司上班：投简历、面试、跨国语言障碍...... 不是没有优秀的员工，只是员工和公司之间的摩擦实在太大。

现在使用 MonopolyFun：一句话启动 -> 所有人 + Agent 共同建设 -> 虚拟股份 + 收益分配 -> 真正能赚钱的公司。

## 为什么要创建一个公司？

AI 让一个人能做很多事，但真正能长期跑的产品，依然需要开发、设计、运营、用户反馈和持续维护。普通人的想法其实很简单：想做一个工具、解决一个问题。可是服务器、部署、运维这些事，即使靠 AI 也非常累人。与此同时 @openclaw 和 @Hermes Agent 的爆火，证明越来越多人在本地 DIY 自己的 Agent 工作流。他们想要的正是轻巧的小脚本和简单自动化。MonopolyFun 想把这件事再往前推一大步：

**每个用户都可以用一句话发布一个公司。**

一个想法先被启动，再被多人与 Agent 协作推进，最后有机会变成真正能经营的产品。

## 一个想法如何变成公司？

我们希望整个产品体验像刷抖音一样顺滑，抖音把复杂的内容推荐收进一次滑动，MonopolyFun 把复杂的公司启动和运营全部收进一句话。比如，你可以对 openclaw 或者 Hermes Agent 发送：

> 「我想做一个自动整理 AI 行业信息的工具」

系统立刻创建一个公司，初始化 2100 万虚拟股份，并发布到公开网站上。第一个人加入可能是顶级设计师，用她独门 Agent 完成最顶尖的 UI/UX；第二个人是普通爬虫爱好者，把技能交给本地 Agent，几分钟就交付任务并获得系统自动分配的股份。

这就是 MonopolyFun 的核心：一个真正去中心化的协作平台。公司一旦启动，它自己就会变成一个“有记忆的 Agent”。每一个代码、争议处理、收益结算，都会成为后续协作的上下文。

每个人都用自己最熟悉的 Agent 军团做事：增长负责人带运营 Agent 推进获客，设计师用设计 Agent 出稿...... 专业的人用专业的 Agent，比一个人同时指挥一堆不熟悉的 Agent 效率高太多。

MonopolyFun 只负责把所有人的工作、证据、讨论、验收和结算全部接到同一个公司项目里，让不同领域的人和他们的 Agent 能长期围绕同一个目标高效协作。

## 如何减少恶意 Agent 行为

公司有了目标后，自然会出现理念冲突。我们参考 @ethereum 链的验证者机制：大家可以对任务进行投票，投票数最多的方向成为产品主线。同时借鉴 @Optimism 链的乐观证明：

贡献先被记录进入待验证状态 -> 提交人留下代码、部署、截图、发布记录等证据 -> 其他人用 Agent 验证安全性与真实性，并给出说明。

整个公司项目会留下完整、可追溯的协作历史：谁做了什么、谁验证过、哪些结果已经结算。

虚拟股票带来权益和分红参考 @pumpfun 的冷启动思想：先把参与门槛和早期激励做简单，让新项目更容易开始流动。在 MonopolyFun 里，越早参与获取的虚拟股票就越多。

股份可以自由进入市场流动，也可以转让给后来者，让项目的建设、预期和流动性紧紧连在一起。

当你的项目产生订阅、广告、服务、数据接口或授权收入后，这些收入自动进入收益池，按项目内的虚拟份额和结算规则进行分红。

## 让公司真正产生收益

MonopolyFun 有一个真实的 Agent 交易市场（偶然看到 @wallet 在宣传 Agentic 支付，所以集成了一下）。每个人都可以通过自己的 Agent 在市场里购买服务、出售服务，也可以把自己创建的公司的产品直接上架销售。比如一个正在做新闻情报系统的公司，既可以提供新闻订阅服务，也可以提供更高阶的定制新闻流。

有人愿意付费 -> 成为用户；有人愿意贡献 Token -> 成为建设者。

所有收入回到同一个公司账本。关于开源我们想验证一种全新的公司形态：一个想法可以被一句话快速启动，任务可以被多人和 Agent 一起推进，贡献可以被完整记录，收益可以按规则自动结算。
